Quickstart

Get from zero to a working request fast.

Step 1

Run the server

git clone https://github.com/honua-io/honua-server.git
cd honua-server
docker compose up -d
curl http://localhost:8080/healthz/ready

Step 2

Make a first API call

curl "http://localhost:8080/rest/services/1/FeatureServer/0/query?where=1%3D1&outFields=*&f=json"

Step 3

Install one SDK

npm install @honua/sdk-js
import { HonuaClient } from "@honua/sdk-js/honua";

const client = new HonuaClient({ baseUrl: "https://your-honua-server.com" });
const result = await client.queryFeatures({ serviceId: "parcels", layerId: 0 });
