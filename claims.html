Public claims matrix

Preview claims stay tied to evidence.

This matrix is the site-owned review surface for public Preview messaging. Each major claim category maps to shipped source evidence, preview proof, an owner ticket, a roadmap/Beta label, or an explicit deferred status before broad outreach.

GTM roadmap source Roadmap Project Docs

Source path tracked by the release lane: honua-sales/docs/strategy/GTM_ROADMAP_PROJECT.md. If that document is private or unpublished, the owner gap remains with honua-sales#4.

Status vocabulary

Review terms used in the matrix.

Source-backed

Claim can be stated as Preview-current when linked evidence supports the exact scope.

Preview partial

Claim can be marketed for evaluation, but partial parity and caveats must stay visible.

Proof pending

Claim may be described as a Preview target, not as validated or generally available.

Roadmap / Beta

Claim belongs to Beta, named pilot, or later roadmap scope.

Deferred

Claim must not be marketed as Preview-ready.

External owner

Release gap belongs to another repo; this site only links the owner ticket or source.
