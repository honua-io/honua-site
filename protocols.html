Protocols

One runtime. Features, maps, raster, analytics, and automation all connect.

Every client you already run — ArcGIS, QGIS, Excel, custom apps, automation tools — connects to a single runtime through the protocol it already speaks. No separate products, no protocol add-ons.

Compatibility First

Open standards and modern interfaces live side by side.

Standards

Legacy standards and raster services ship as core

GeoServer-facing standards, QGIS integrations, rendered maps, and ArcGIS-compatible raster/image workflows are built in, not bolted on.

  • WFS 2.0 transactions ship at launch alongside metadata and query operations.
  • WMS 1.3, WMTS 1.0, and OGC API Maps all have published conformance results.
  • GeoServices MapServer and ImageServer are included alongside the open standards surface.

Modern Access

gRPC and MCP on the same runtime

Typed app integrations and structured tool access run on the same runtime as the compatibility and standards layers.

  • gRPC fits app backends, service integrations, and mobile workflows.
  • MCP exposes discovery, schema inspection, and filtered query tools.
  • SDKs and automation coexist with legacy GIS clients — no separate infrastructure.

Coverage

What each protocol unlocks.

GeoServices REST

Keep your Esri clients running

FeatureServer (query, edit, attachments, related records, replicas), MapServer (export, identify, legend, find), ImageServer, and Geometry Service (buffer, intersect, project, union, and more).

WFS 2.0

Move off GeoServer without breaking clients

GetCapabilities, DescribeFeatureType, GetFeature, GetPropertyValue, and Transaction ship at launch, alongside SLD import/export and GeoServer config migration.

OGC API

Full CITE conformance

OGC API Features with transactions and CQL2 filtering, OGC API Tiles for vector and raster, and OGC API Maps for server-rendered images. All CITE tests passing.

OData v4

Spatial data in business tools

Full CRUD with $filter, $select, $orderby, $top, $skip, $count, $search, $apply, $batch, and spatial functions. Connect Excel, Power BI, Tableau, and ERPs directly.

Raster + tiles

Maps and imagery your clients expect

ImageServer, OGC API Maps, WMS, WMTS, vector tiles, TileJSON, and rendered PNG/JPEG/TIFF outputs alongside feature APIs.

Cloud-native formats

Modern data movement

GeoParquet, FlatGeobuf, PMTiles, and Cloud Optimized GeoTIFF support import, query, export, and distribution workflows.

gRPC

Faster apps, lower bandwidth

Typed binary protocol for high-throughput queries, batch edits, and streaming. Unary access in base platform; streaming in Pro/Enterprise.

MCP

Connect AI agents and automation

Service discovery, schema inspection, and filtered query workflows for AI agents and automation tools through a public MCP package.
