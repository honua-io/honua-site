How execution is gated

The OGC API Processes catalog and schemas are public and load anonymously here. Execution (POST /execution, jobs) requires a server X-API-Key; the public demo has no anonymous execution key, so a live submit returns an honest 401 Authentication required. The Pro analytics.* processes additionally require a Pro entitlement (HTTP 402) once authenticated. This demo surfaces the live gate rather than faking a result.
