// what's in each tier

Community, Pro, Enterprise — by capability.

This is the edition gating that ships in the product today, drawn from the in-product feature catalog — not a marketing wishlist. Pricing is still being shaped with early customers, and the boundaries can still move as the tiers are finalized. Each tier includes everything in the tier to its left.

Community · free · ELv2

Source-available — free to read, serve, and run yourself.

  • All OGC & GeoServices read, query, tiles, and metadata surfaces
  • File import — GeoJSON, Shapefile, GeoPackage
  • ArcGIS Portal token issuance + read-only Portal sharing facade (Esri-client interop)
  • Smart geometry-aware style defaults
  • Temporal query filtering + extent discovery
  • COG import / export
Pro · pricing TBD

Everything in Community, plus editing, operational, analytics, and migration tooling.

  • Multi-user editing via FeatureServer — applyEdits / add / update / delete
  • Geofence alerts (enter/exit) + evaluation engine · webhook delivery
  • Forward / reverse geocoding + provider failover
  • Output caching + Redis distributed cache (multi-node)
  • Real-time feature streams — WebSocket / SSE with replay cursors
  • High-DPI, large-dimension, and rich-overlay static maps
  • Auto-cartographic styling
  • COG serving from S3 / Azure + temporal raster mosaic
  • Spatial analytics — clustering, spatial join, buffer aggregate, density, sun/shadow, slice, line-of-sight, viewshed
  • Temporal animation — date-bin histograms, time-series tiles, animation API
  • PDF print output + full layout templates
Enterprise · pricing TBD

Everything in Pro, plus the enterpriseready.io baseline and heavy-integration features.

  • SSO — OIDC multi-provider (Azure AD, Google, generic) + claim-to-role mapping
  • Dwell + attribute-threshold alert triggers
  • Alert channels — email, Slack, Microsoft Teams, AWS SNS, Azure Event Grid, digest
  • Batch geocoding
  • Source migration import — ArcGIS REST (GeoServices) + GeoServer
  • Esri-style branch versioning — named gdb versions, reconcile/post, VersionManagementServer (PostgreSQL)
