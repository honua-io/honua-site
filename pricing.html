How to use it

Clone, run, ship.

Honua is designed to be self-hostable end to end. No license key, no phone-home, no call required before you can evaluate.

Get it running

Local quickly, cloud through tracked modules.

Start with the Docker quickstart, point a standards client at it, confirm your data flows. When you're ready, the Terraform modules target ECS/Fargate, Lambda, EKS, Azure Container Apps, Functions, or AKS once validation evidence is available.

Keep it running

Use Preview SDKs and the MCP schema deliberately.

Pick the SDK source path that matches your stack and check its release channel. Plug Honua into MCP-aware agents for Preview evaluation, and wire OpenTelemetry into your existing dashboards.
