Open Core

Self-host the platform. Talk to us when you want help.

Honua is open core. The runtime, SDKs, protocol specs, mobile stack, and Terraform modules are all public — clone a repo, read the code, run it yourself. Engage commercially when you need support, migration delivery, or a hand in production.

How to use it

Clone, run, ship.

Honua is designed to be self-hostable end to end. No license key, no phone-home, no call required before you can evaluate.

Get it running

Local in minutes, cloud in an afternoon.

Start with the Docker quickstart, point a standards client at it, confirm your data flows. When you're ready, the Terraform modules take you to ECS/Fargate, Lambda, EKS, Azure Container Apps, Functions, or AKS.

Keep it running

Use the SDKs and MCP schema anywhere.

Pick the SDK that matches your stack. Plug Honua into your existing agents via MCP. Wire OpenTelemetry into the dashboards you already run. None of it requires permission from us.
