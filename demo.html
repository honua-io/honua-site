    Every byte of data on this map came through the SDK.

    MapLibre GL draws the pixels; the open-source Honua JS SDK does the talking — server compatibility checks, layer metadata probes, GeoServices feature queries, and the click-to-inspect hit tests. This is the same client you would ship in your own app. 

    // the demo page's data path, condensed
const client = new HonuaSDK.HonuaClient({ baseUrl: "https://demo.honua.io" });

const dataset = HonuaSDK.createDataset({
  id: "maui", client,
  sources: [{
    id: "parcels",
    protocol: "geoservices-feature-service",
    locator: { url: "https://demo.honua.io", serviceId: "maui-parcels", layerId: 0 },
    capabilities: HonuaSDK.PROTOCOL_DEFAULT_CAPABILITIES["geoservices-feature-service"],
  }],
});

// click → attribute popup
const { features } = await dataset.source("parcels").query({
  spatialFilter: HonuaSDK.envelope(west, south, east, north, { wkid: 4326 }),
  outFields: ["*"], returnGeometry: false, pagination: { limit: 1 },
});
