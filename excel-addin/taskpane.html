Honua for Excel

preview
This taskpane is meant to run inside Excel. Open Excel, sideload the Honua for Excel add-in (see the README), and launch it from the ribbon. You can still browse layers below, but loading into a sheet needs Excel.

Up to 1,000 rows load as an Excel table. Attributes are editable; geometry is shown read-only.
