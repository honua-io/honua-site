// what this shows

  • One query, one result shape. The Honua JS SDK's protocol-neutral contract — Dataset → Source → Query → Result — treats the protocol as a field on a source descriptor, not an architecture decision.
  • GeoServices RESTCommunity
    The same endpoints ArcGIS clients speak — /rest/services/…/FeatureServer/0/query.
  • OGC API FeaturesCommunity
    The open standard, first-class — /ogc/features/collections/…/items with CQL2 filters.
  • OData v4Community
    The same rows your spreadsheet and BI tools can reach — /odata/…?$filter=.

Every protocol surface — read, query, tiles, metadata — ships in the free Community edition (ELv2). That is the point: when every door is open and standard, the data is yours to walk out with.
