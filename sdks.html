Protocol Clients

Query features, render maps, stream data.

Each language SDK includes typed clients for FeatureServer, MapServer, OGC API, and optional gRPC streaming. Pick the language that fits your stack.

Migration

Replace ArcGIS web apps without a full rewrite.

JavaScript SDK

Scanners, codemods, and compatibility wrappers.

The JavaScript SDK has dedicated entry points for Esri migration: usage scanners find what your apps depend on, codemods automate safe replacements, and compatibility wrappers let existing ArcGIS code run against Honua endpoints during the transition.

  • Esri FeatureLayer, MapImageLayer, and widget compat wrappers.
  • Migration CLI with scanning, codemods, and fixture gating.
  • Parity matrices and reconciliation helpers.

Admin Clients

Manage the server from any language.

Admin clients are generated from the same OpenAPI contract across JavaScript, .NET, and Python. Publish services, manage layers, configure styles, apply manifests, and automate server operations.

  • Service, layer, connection, and style management.
  • Manifest apply with dry-run for GitOps workflows.
  • Capability detection via runtime handshake.

Mobile

Field apps with offline sync and native maps.

The .NET MAUI mobile SDK handles offline GeoPackage storage, background sync, dynamic forms, photo/video capture, and gRPC-first transport. iOS, Android, and Windows from one codebase.

The MCP server package gives AI agents and automation tools a structured interface for service discovery, schema inspection, and filtered queries — without raw API access.

Licensing and Compatibility

All SDKs are Apache 2.0. All version together.

Open source

Every SDK — protocol clients, admin clients, mobile toolkit, and MCP — is Apache 2.0. Evaluate, fork, and ship without licensing friction.

Versioned together

Server and SDK versions publish compatibility contracts so teams know exactly what can upgrade together. Admin clients are generated from the same OpenAPI contract across all three languages.
