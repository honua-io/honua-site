More Than Libraries

Migration tooling, data services, and automation built in.

Esri migration toolkit

The JavaScript SDK includes usage scanners, safe codemods, compat wrappers, and reconciliation helpers for teams replacing ArcGIS web applications.

MCP server package

Give assistants and automation tools access to Honua through a dedicated MCP package for service discovery, schema inspection, and filtered queries.

Data-service access

Geocoding, routing, and the broader data-service surface ship through the same SDK family instead of becoming a separate product line.

Versioning and compatibility

Server and SDK versions publish compatibility contracts so teams know exactly what can upgrade together.

Quickstarts

Install one SDK, make one authenticated call, and see where to go next. No hunting across repos.
