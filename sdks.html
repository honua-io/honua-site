SDKs

Official SDKs for JavaScript, .NET, Python, and agent workflows.

Honua gives developers typed clients, migration tooling, agent access, compatibility guidance, and direct install paths.

Primary SDKs

Choose the SDK that matches how your team builds.

JavaScript

Web apps and migrations

The JavaScript SDK covers typed clients, OGC usage, Esri compatibility helpers, migration scanning, and runtime parity matrices.

npm install @honua/sdk-js

.NET

Enterprise services and MAUI apps

The .NET SDK focuses on gRPC queries, admin workflows, and geocoding, with a natural path into MAUI and broader Microsoft stacks.

dotnet add package Honua.Sdk.Grpc --prerelease
dotnet add package Honua.Sdk.Admin --prerelease

Python

Analysis, automation, and data workflows

The Python SDK gives data teams feature access, geocoding, admin automation, and optional gRPC support for streaming workflows.

pip install honua-sdk
pip install honua-sdk[grpc]

Beyond Basic SDKs

More than client libraries.

Migration toolkit

The JavaScript SDK includes scanners, codemods, compat wrappers, and reconciliation helpers for organizations replacing ArcGIS-centric applications.

MCP server package

Use Honua with AI assistants and coding agents through a dedicated MCP package for discovery and query workflows.

Versioning and compatibility

Server and SDK versions publish compatibility expectations so teams know what can upgrade together.

Quickstarts

Quickstarts let a team install one SDK, make one authenticated call, and understand where to go next without hunting across repos.

How To Choose

Choose the right SDK for your team.

If your team is... Start here Why
Replacing ArcGIS-heavy web apps JavaScript SDK Compat wrappers and migration tooling lower the cost of translation.
Building enterprise services or MAUI apps .NET SDK gRPC and admin clients align well with Microsoft and mobile-adjacent stacks.
Automating data workflows or analysis Python SDK Simple feature access and optional gRPC support fit notebooks, scripts, and pipelines.
Wiring AI assistants or agents into data discovery MCP server Discovery, schema inspection, and filtered queries become tool-safe and explainable.
