Surface How Honua supports it What it unlocks
OGC and classic services OGC API Features, Tiles, Maps; WFS, WMS, WMTS, and WCS. ArcGIS Pro, QGIS, web maps, partner systems, and standards-based clients.
Esri compatibility GeoServices REST compatibility for ArcGIS-facing workloads. ArcGIS Pro, ArcGIS Online, Enterprise, JS API, and systems already built around Esri service URLs.
Files and object data GeoJSON, GeoPackage, Shapefile, FlatGeobuf, GeoTIFF, COG, STAC, MVT, CSV with geometry, and GeoParquet. Data exchange, field workflows, archives, tiles, rasters, and analytics pipelines.
Databases PostGIS/PostgreSQL-first workflows, cloud PostgreSQL variants, and SQLite/GeoPackage paths. Use existing spatial data stores without forcing a full data migration first.
Modern protocols gRPC/Protobuf for typed high-throughput APIs and MCP for agent-readable spatial operations. New apps, SDKs, automation, and AI workflows without scraping maps or guessing schemas.
